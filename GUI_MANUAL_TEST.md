# ChatScreen2PDF v0.3.5 GUI 手动验收清单

## 前置条件
- Python 3.10+, FFmpeg, pip install -r requirements.txt, 至少一个 .mp4 测试视频

## 验收项

### 界面中文化
- [ ] 窗口标题: ChatScreen2PDF v0.3.5
- [ ] 所有界面文字为中文
- [ ] 日志面板全中文，无 Checking FFmpeg / Scanning 等英文
- [ ] 日志面板不出现 img2pdf DEBUG / pikepdf / PIL 底层日志
- [ ] FFmpeg 检查只显示一次: "正在检查 FFmpeg..."

### 输入方式
- [ ] 选择文件夹后视频列表填充
- [ ] 添加视频文件后视频列表填充
- [ ] 添加视频文件后不会扫描项目目录
- [ ] pipeline 日志显示"使用已选择的视频文件: N 个"

### 默认输出路径
- [ ] 选择输入文件夹后，输出目录自动设为输入文件夹
- [ ] 添加视频文件后，输出目录自动设为视频所在文件夹
- [ ] 手动选择输出目录后不再自动覆盖

### 裁剪模式
- [ ] 点击"选择裁剪区域"前拖拽不产生裁剪框
- [ ] 点击"选择裁剪区域"后拖拽一次生成裁剪框
- [ ] 释放鼠标后裁剪框固定
- [ ] 清除裁剪后坐标和框都消失

### 进度与状态
- [ ] 开始前状态显示"等待开始"
- [ ] 进度从 0/N 开始
- [ ] 每完成一个视频进度更新
- [ ] 单视频: 0/1 -> 1/1
- [ ] 底部状态文本显示当前阶段

### 参数日志
- [ ] 开始时日志显示中文参数

### 打开输出文件夹
- [ ] 转换前"打开输出文件夹"按钮禁用
- [ ] 转换成功后"打开输出文件夹"按钮启用
- [ ] 转换失败后"打开输出文件夹"按钮也启用
- [ ] 点击后打开 PDF 所在目录
- [ ] 输出目录不存在时弹出中文错误提示

### 错误处理
- [ ] 不选择视频时点击开始，弹出中文提示
- [ ] 不选择输出目录时自动设置

### 日志文件
- [ ] 日志文件固定生成在项目 logs/ 目录

### PDF 生成
- [ ] 普通 PDF 正常生成
- [ ] 中文文件名正常

### v0.3.5 新增：输出覆盖保护
- [ ] 输出目录已有同名 PDF 时，自动生成 `_1` 后缀文件
- [ ] 连续同名时依次生成 `_1`、`_2`、`_3`
- [ ] 日志显示最终输出文件名（如：文件已存在，自动改名为：1_1.pdf）

### v0.3.5 新增：OCR 隐藏
